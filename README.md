# 🚗 Uber SQL Capstone Project

## 📌 Overview
This project analyzes Uber ride-sharing data using SQL.

## 🔧 Features
- Data Cleaning
- City Performance Analysis
- Revenue Leakage Detection
- Cancellation Analysis
- Seasonal Trends
- Index Optimization
- Views & Triggers

## 🛠 Tools Used
- MySQL
- SQL

## 📊 Key Insights
- High cancellations during morning hours
- Revenue leakage due to missing payments
- Certain cities require more drivers
